## 🚀 HOW TO GET YOUR LINKS

### **Step 1: Create GitHub Repository**

1. Go to https://github.com
2. Click the **"+"** button (top right)
3. Click **"New repository"**
4. Repository name: `rootwise`
5. Description: `Ancestral herbal wisdom platform - 150 medicinal herbs`
6. Choose **Public**
7. Click **"Create repository"**

Your repo URL will be:
```
https://github.com/YOUR_USERNAME/rootwise
```

---

### **Step 2: Upload Files**

**Option A: GitHub Web Interface (Easiest)**

1. On your new repo page, click **"uploading an existing file"**
2. Drag and drop all files:
   - rootwise-enhanced.html
   - INDEX.md
   - README.md
   - LICENSE
   - .gitignore
   - All other files
3. Commit message: `Initial commit: Rootwise v1.1 with 150 herbs`
4. Click **"Commit changes"**

**Option B: Git Command Line**

```bash
# In your rootwise folder
git init
git add .
git commit -m "Initial commit: Rootwise v1.1 with 150 herbs"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/rootwise.git
git push -u origin main
```

---

### **Step 3: Enable GitHub Pages**

1. Go to your repository
2. Click **"Settings"** tab
3. Scroll to **"Pages"** (left sidebar)
4. Under **"Source"**:
   - Branch: `main`
   - Folder: `/ (root)`
5. Click **"Save"**
6. Wait 1-2 minutes

Your app will be live at:
```
https://YOUR_USERNAME.github.io/rootwise/rootwise-enhanced.html
```

## 🔧 CUSTOM DOMAIN (Optional)

Want to use your own domain like `rootwise.app`?

### **Setup:**

1. Buy domain from Namecheap, Google Domains, etc.
2. Add these DNS records:
   ```
   Type: A
   Name: @
   Value: 185.199.108.153

   Type: A  
   Name: @
   Value: 185.199.109.153

   Type: A
   Name: @
   Value: 185.199.110.153

   Type: A
   Name: @
   Value: 185.199.111.153

   Type: CNAME
   Name: www
   Value: YOUR_USERNAME.github.io
   ```

3. In GitHub repo Settings → Pages:
   - Enter your custom domain
   - Check "Enforce HTTPS"

4. Wait 24-48 hours for DNS propagation

Your app will be at: `https://rootwise.app` 🎉
