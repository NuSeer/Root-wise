# 🚀 ROOTWISE - QUICK START GUIDE

Get Rootwise running in under 5 minutes!

---

## 📦 What You Need

**Nothing!** Rootwise is pure HTML/CSS/JavaScript. No installation, no build process, no dependencies.

---

## ⚡ 3 Ways to Run Rootwise

### Option 1: Double-Click (Easiest)

1. Download `rootwise-enhanced.html`
2. Double-click the file
3. Opens in your default browser
4. ✅ Done!

---

### Option 2: Local Server (Recommended)

```bash
# Python 3
python -m http.server 8000

# Then visit: http://localhost:8000/rootwise-enhanced.html
```

---

### Option 3: GitHub Pages (Free Hosting)

1. Push code to GitHub
2. Enable Pages in Settings
3. Live at: `yourusername.github.io/rootwise`

---

## 🎯 First Steps

1. Open `rootwise-enhanced.html`
