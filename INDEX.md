## 🔧 CUSTOMIZATION

### **Change Colors**
Edit CSS variables in the HTML file:
```css
:root {
    --forest-dark: #1a3a26;
    --sage-green: #6b9478;
    /* Change these to your brand colors */
}
```

### **Add More Herbs**
Edit the `HERBAL_DATABASE` array:
```javascript
const HERBAL_DATABASE = [
    {
        "name": "Your Herb",
        "uses": "Health benefits",
        // ... more fields
    },
    // ... existing herbs
];
```

### **Modify Features**
All JavaScript is in the HTML file - easy to edit!
